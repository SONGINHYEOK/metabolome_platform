# 시연 영상 시나리오

## 영상 개요
- **목적**: 발표평가용 프로토타입 데모
- **예상 시간**: 3~5분
- **URL**: http://127.0.0.1:6321/

---

## Scene 1. 랜딩페이지 (30초)

**화면**: `/`

**나레이션**:
> "특용작물 대사체 통합 디지털 플랫폼은 2-Track 구조로 설계되었습니다. 80종 대사체 DB 구축을 목표로, 연구자용 Research Portal과 대국민용 Public Portal을 분리하여 운영합니다."

**동작**:
1. 랜딩페이지 전체 화면 보여주기
2. 상단 수치 카드 3개 강조 (80종 / 50종 / 2-Track)
3. 하단 2-Track 선택 카드에 마우스 호버 (애니메이션 확인)
4. **RESEARCH PORTAL** 클릭

**화면**: `/research/`

**나레이션**:

**동작**:
1. 네이비 헤더, 브레드크럼 확인
2. Quick Stats (등록 성분, 분석 작물) 보여주기
3. 4개 기능 카드 훑어보기 (ACTIVE / 개발 예정 구분)
4. **Compound Catalog** 카드 클릭

---

## Scene 3. Compound Catalog (1분 30초) — 핵심

**화면**: `/research/catalog/`

**나레이션**:
> "Compound Catalog은 3-컬럼 레이아웃으로, 좌측 필터 패널에서 작목, 부위, 산지, 연도, QC 상태를 조건 기반으로 탐색할 수 있습니다."

### 동작 1 — 전체 레이아웃 (20초)
1. 3-컬럼 구조 전체 보여주기 (필터 | 목록 | 상세)
2. 자동 선택된 첫 번째 성분의 상세패널 확인

### 동작 2 — 필터링 (20초)
1. 좌측에서 작목 "인삼" 선택 → 목록이 인삼 성분만 표시됨
2. "총 16건 중 6개 표시" 변경 확인
3. 필터 초기화 클릭

### 동작 3 — 성분 선택 (20초)
1. 목록에서 **Ginsenoside Rg1** 클릭 → 우측 상세패널 갱신
2. L1 배지, IN-HOUSE 배지, Score 96, Similarity 0.94 확인
3. 메타 정보 (작목, 부위, 산지, 연도) 확인

### 동작 4 — 상세 패널 (30초)
1. 탭 UI 보여주기 (개요 활성, 나머지 개발 예정)
2. 활용모델 라디오 버튼 (원산지 판별 / 성분 패턴 분석)
3. TOP 3 CONTRIBUTION 바 차트 (Saponin 65%, Sugar 25%, Aglycone 10%)
4. QUICK ANALYSIS — 원산지 판별 결과 **HIGH**, SCORE 96, SIMILARITY 0.94
5. 다른 성분 (**Decursin** 등) 클릭하여 상세패널 동적 변경 확인
6. 하단 스크롤 → 전문가용 근거 분석 CONCEPT IMAGE (A/B/C 카드)

**나레이션 마무리**:
> "각 성분별 Annotation Level, Library Match 정보, 종합 Score를 근거 기반으로 제공하며, MSI 레벨 체계에 따라 동정 신뢰도를 투명하게 표시합니다."

## Scene 4. 대국민 포털 메인 (20초)

**화면**: `/public/` (브라우저 뒤로가기 → 랜딩 → PUBLIC PORTAL 클릭)

**나레이션**:
> "대국민 포털은 소비자, 농업인, 지자체를 대상으로 쉬운 언어의 성분 정보와 비교 분석을 제공합니다."

**동작**:
1. 50종 DB 구축 / 80종 확장 배지 확인
2. 대상별 카드 (소비자/농업인/지자체) 보여주기
3. 4개 기능 카드 훑어보기
4. **작목 상세 대시보드** 클릭

---

## Scene 5. 작목 상세 대시보드 (1분 30초) — 핵심

**화면**: `/public/dashboard/`

### 동작 1 — 환경 지표 (20초)

**나레이션**:
> "작목 상세 대시보드에서는 공공 API 연계 지역·환경 지표를 확인할 수 있습니다. 연평균 기온, 강수량, 토양질 지수를 API 기반으로 제공합니다."

1. 지역·환경 지표 카드 3개 강조 (12.8°C / 1150mm / A등급)
2. "API 기반" 배지 확인

### 동작 2 — 차트 (20초)

**나레이션**:
> "성분 분포 요약 차트에서 인삼의 주요 성분 Score를 확인할 수 있으며, 비교 결과 분석 차트에서 인삼과 황기의 성분별 차이를 시각적으로 비교합니다."

1. 성분 분포 바 차트 보여주기
2. 비교 결과 그룹 바 차트 (인삼 파란색 vs 황기 주황색)

### 동작 3 — 비교 변경 (20초)
1. 좌측 사이드바에서 TARGET B를 "당귀"로 변경
2. "비교 분석 적용" 버튼 클릭
3. 차트가 인삼 vs 당귀로 갱신되는 것 확인
4. 환경 지표도 해당 지역으로 변경 확인

### 동작 4 — AI 해설 및 산출물 (30초)

**나레이션**:
> "우측 패널에서는 AI 기반 요약·비교·해설을 제공하며, 정책·산업·농업인별 활용 시나리오를 안내합니다. 불확실성 및 해상도 안내를 통해 데이터의 신뢰성 범위도 투명하게 표시합니다."

1. 우측 AI 해설 콘텐츠 읽기
2. 활용 시나리오 3개 (정책/산업/농업인) 보여주기
3. 활용모델 (원산지 확인, 품질 선택, 환경 해석)
4. 불확실성 안내 박스 확인
5. 대국민 산출물 미리보기 카드 보여주기
6. 하단 OUTPUT DOWNLOADS 버튼들 (요약카드, 비교카드, PDF 리포트)

## 촬영 팁

- **해상도**: 1920x1080 (Full HD) 권장
- **브라우저**: Chrome, 전체화면 또는 탭만 녹화
- **마우스**: 천천히 이동, 클릭 포인트에서 1초 정도 멈춤
- **전환**: 페이지 이동 시 브레드크럼 활용하면 자연스러움
- **녹화 도구**: OBS Studio 또는 macOS 기본 화면 녹화 (Cmd+Shift+5)
